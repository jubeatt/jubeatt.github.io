Why？

當放在 <head> 中，執行到 JavaScript 時，下面 <body> 內容還沒有產生，所以當然抓不到任何東西。

但是放在 </body> 前，執行到 JavaScript 時，內容已經產生好了，這時候就抓的到東西。

這也就是為什麼大家都習慣把 JavaScript 寫在 </body> 前的原因。
