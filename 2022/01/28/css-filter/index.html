CSS 好用的 filter 屬性

1月 28 2022 CSS

加上 filter 後質感立馬提升！

列表

主要會用在圖片上的屬性（但不僅限於圖片），簡單來說就是濾鏡，看範例最快！

filter: brightness() 設定明亮度

filter: blur() 設定模糊

filter: contrast() 設定對比度

filter: drop-shadow() 設定陰影

filter: hue-rotate 色調旋鈕
