因為自己用到 useContext 的頻率不高，所以在用的時候都用比較正統的方式來撰寫。

到了現在才發現原本有一種更好的寫法，可以把邏輯交給 Context 自己來處理，而不是寫在根元素上。總之這是個蠻不錯的練習，所以想記錄一下。
