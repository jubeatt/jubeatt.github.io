補充：每個 props 的用途

  • isAnimating: boolean 是否顯示進度條
  • animationDuration: number 從開始到結束的時間 (ms)
  • incrementDuration: number 進度條遞增的間隔時間（ms）
  • minimum: number 進度條的起始點（0~1）
