設定 SSL 憑證

3月 9 2022 Network

結果也沒那麼複雜。

簡述

本來想說想說 SSL 要錢，加上我懶所以沒打算弄。不過後來看到 同學寫的紀錄 裡面正好提到：

有一個網站叫做 lets encrypt 可以免費頒發 SSL 的證書，而 certbot 就是可以幫你自動連到 lets encrypt 申請證書，甚至在過期的時候也會自動幫你再申請！！簡直是 nice 到不行

所以就來幫自己的網站補上 SSL 憑證了。

步驟

官網其實就有 教學步驟 了，不過保險起見還是寫一下好了：

  1. 安裝 snapd（EC2 的 Ubuntu 已經有內建，所以這步略過）

  1. 更新 snapd：sudo snap install core; sudo snap refresh core
  2. 安裝 Certbot：sudo snap install --classic certbot
  3. 確認 Cerbot 指令可以用：sudo ln -s /snap/bin/certbot /usr/bin/certbot
  4. 設定要使用 SSL 的網址：sudo certbot --apache

  1. 重新啟動 Cerbot：sudo certbot renew --dry-run

最後就能看到有鎖頭囉！

補充－後來新增的子網域也想加上 SSL 的話

加上 -d <新的子網域>：

這樣就行囉。不確定不加的話是不是會自動跳選項給你選，之後有機會再試看看吧。
