學到的事情

雖然 PHP 寫起來很直覺，但當功能增加時就會變得很複雜。光是實作上面的功能就已經開了21 個 PHP 檔，如果眼睛不夠利的話很常會搞混哪個檔案在做什麼，是件蠻困擾的事情。

學會與雜亂的 code 相處

我想身為工程師必須學會的事情，code 再多再長，也是要想辦法找出你的目標，不能因為 code 長就眼花撩亂。
