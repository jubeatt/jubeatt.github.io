這裡會拿這隻 API 來示範。

主要是有些參數容易搞混，這邊先說明清楚：

  • dataType 預期 server 要回傳的資料格式，沒指定的話就看 server 怎麼設定。
  • contentType 要傳給 server 的資料格式，沒指定的話會是 application/x-www-form-urlencoded; charset=UTF-8
