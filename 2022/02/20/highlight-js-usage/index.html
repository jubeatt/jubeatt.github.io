引入方式

highlight.js 主要就兩隻檔案：

  • 每個主題的 CSS 檔案
  • 渲染程式碼區塊的 JavaScript 檔案

這邊我只列幾個比較方便的，詳細請直接參考這裡。

  1. CDN 引入

像 default.min.css 可以換成自己喜歡的主題，例如：agate.min.css

如果不知道有些主題的檔名是什麼，建議直接下載檔案來看就知道了（像 A 11 Y DARK 的檔名是 a11y-dark.min.css）。
