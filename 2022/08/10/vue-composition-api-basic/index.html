8月 10 2022 Vue

希望解決 Options API 的弊病

  • code 散落各地，缺發群聚性，可讀性也差
  • 同樣的邏輯沒辦法被重用

簡單來說 Composition API 可以讓你的 code 群組起來，也實現了「把邏輯抽出去」的功能，換句話說就跟 React 的 custom hook 是很像很像的概念，所以才會這麼多人愛用它。
