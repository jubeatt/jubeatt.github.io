React 的第五個 hook：memo

4月 7 2022 React

跟效能有關的 hook 之一。

簡述

把 Component 給記起來，如果 props 沒有變就不會 re-render

一般在父元件的 state 改變時，會連帶底下的所有子元件 re-render，可是有些時候不需要阿，子元件根本就沒改到什麼內容，幹嘛 re-render？

這時候就可以用 memo，來對子元件做一層包裝，讓它只有在 props 改變時才會跟著 re-render
