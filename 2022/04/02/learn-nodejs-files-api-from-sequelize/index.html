這篇學完後我應該要學到：

  1. __dirname，目前位置的資料夾路徑
  2. __filename，目前位置的檔案路徑
  3. path.basename，讀取路徑最後的資料夾 / 檔案的名稱
  4. fs.readdirSync，讀取資料夾底下的所有檔案 / 資料夾
  5. path.join 拚路徑遊戲，填什麼就接什麼
